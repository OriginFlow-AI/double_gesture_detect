# Gemini 335 六面标定推荐

## 已确认设备

```text
厂家：Orbbec
型号：Gemini 335
USB ID：2bc5:0800
当前设备序列号：CP0HC5300092
视频节点：/dev/video0 - /dev/video7
```

推荐配置位于：

```text
configs/gemini335_six_face_calibration.json
```

## 厂家固定参数

Orbbec Gemini 330 Series Datasheet V1.6 给出的 IMU 参数：

- 六轴 IMU：三轴加速度计和三轴陀螺仪。
- X 轴向右、Y 轴向下、Z 轴向相机前方。
- 加速度量程：`±4g`，即约 `±39.2 m/s²`。
- 陀螺仪量程：`±1000 dps`，即约 `±17.45 rad/s`。
- 支持采样率：`50 / 100 / 200 / 500 / 1000 Hz`。
- Gemini 335 推荐固件：`1.6.00`。
- 工作温度：15 FPS 时 `-10～45℃`；30/60 FPS 时 `-10～40℃`。

厂家规格没有提供可复制到所有设备的六面零偏和比例因子。它们是当前设备、安装应力及温度相关量，必须根据六面实测数据求解。

## 推荐采样参数

```text
采样率：200 Hz
预热：60 秒
每面静置：5 秒
每面采样：10 秒 / 2000 点
统计：去掉两端各 10% 后计算截尾均值
标准重力：9.80665 m/s²
```

六个姿态以“哪个坐标轴垂直向上”命名。相机正面朝上对应 `+Z up`，背面朝上对应 `-Z up`。

## 计算

每个轴分别使用正、负两面的稳态均值：

```text
bias = (positive_mean + negative_mean) / 2
scale = 2g / (positive_mean - negative_mean)
corrected = (raw - bias) * scale
```

陀螺仪使用六面所有静止样本估计零偏：

```text
gyro_corrected = gyro_raw - gyro_bias
```

六面静态标定不能计算陀螺仪比例因子，也不能给出绝对航向角。

## 使用原则

1. 优先通过 Orbbec SDK v2 读取设备出厂 IMU intrinsic。
2. 先记录 SDK 实际输出和出厂内参，不默认在业务层再次手工应用厂家矩阵。
3. 只有确认 SDK 输出是未校正原始值后，才考虑手工应用出厂内参。
4. 六面结果先用于验证；确认存在稳定残差后，最多叠加一次残差修正。
5. 不要把出厂内参和同一套六面修正重复应用。
6. 当前工程的 `pitch/roll/yaw` 门限不是六面标定系数。
7. Gemini 335 的公开规格没有磁力计，六面标定不会消除 yaw 漂移。

当前只确认了设备型号、USB ID 和序列号，尚未通过 Orbbec SDK 读取设备实际固件版本。`1.6.00` 是厂家推荐版本，不是本机已验证版本。

## 验收

配置文件将采样姿态、静止判定和结果阈值分开保存。推荐上线门槛：

- 六面均达到至少 95% 有效样本；
- 修正后每面的主轴误差不超过 `0.15 m/s²`；
- 修正后重力模长 RMSE 不超过 `0.10 m/s²`；
- 任一比例因子超出 `0.95 - 1.05` 时检查摆放、单位和是否重复修正；
- 陀螺仪静止零偏任一轴超过 `0.03 rad/s` 时重新预热并采样。

这些验收阈值属于工程推荐，不是厂家保证值。

## 厂家值与工程值对比

| 参数 | 厂家值 | 配置采用值 | 取舍 |
|---|---:|---:|---|
| 加速度量程 | ±4g、约 ±39.2 m/s² | 原样保存 | `4 × 9.80665 = 39.2266`，与厂家值只差四舍五入 |
| 陀螺仪量程 | ±1000 dps、约 ±17.45 rad/s | 原样保存 | 精确换算为 `17.453293 rad/s`，运行配置遵从厂家标称 |
| 采样率 | 50/100/200/500/1000 Hz | 200 Hz | 静态标定不需要 500/1000 Hz；200 Hz 可在 10 秒内获得 2000 点 |
| 工作温度 | 15 FPS：-10～45℃；30/60 FPS：-10～40℃ | 按帧率分别保存 | 当前工程运行 30 FPS，因此上限按 40℃理解 |
| 标准重力 | 厂家未指定标定常数 | 9.80665 m/s² | 使用国际标准重力，六面计算需要统一参考值 |
| 预热时间 | 未公布 | 60 秒 | 降低上电初期温漂；属于保守工程建议 |
| 每面稳定等待 | 未公布 | 5 秒 | 留出人工翻面后的振动衰减时间 |
| 每面采样时长 | 未公布 | 10 秒 | 在重复性与操作总时长之间折中 |
| 截尾比例 | 未公布 | 两端各 10% | 抑制翻面余振、偶发 USB/调度异常样本 |
| 姿态误差窗口 | 未公布 | 模长 ±5%，横轴 ≤0.1g | 约允许 5.7° 摆放误差，用作采样筛选，不是最终精度 |
| 修正后单面误差 | 未公布 | ≤0.15 m/s² | 初始目标，约 15.3 mg；首轮实测后再调整 |
| 修正后模长 RMSE | 未公布 | ≤0.10 m/s² | 初始目标，约 10.2 mg；不是厂家保证 |
| 陀螺仪静止偏置 | 未公布 | ≤0.03 rad/s | 约 1.72 dps，仅作初筛，实际应根据静态分布收紧 |

厂家没有公开 Gemini 335 IMU 的零偏、噪声密度、比例因子误差和轴间不正交容差。因此配置中的误差门槛不能与厂家“合格线”直接比较，只能先用于发现摆放错误、单位错误、运动样本和明显异常设备。
